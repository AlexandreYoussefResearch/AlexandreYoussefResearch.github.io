Scientific Computing Project – EPFL

Parallelized Conjugate Gradient Solver
Developed during a high-performance computing course at EPFL. This project implements and compares two parallel solvers for large, sparse, symmetric positive-definite systems:
- A distributed-memory MPI-based CPU version
- A GPU-accelerated CUDA version with custom kernels
Both versions are benchmarked and analyzed across multiple dimensions of scalability and performance.
