3D Numerical Simulations with Vortex Penalization Methods

Efficient and Accurate Simulations of Three-Dimensional Flows Past Obstacles with Vorticity-Based Penalization Methods
Conducted jointly at the Van Rees Lab (MIT) and the Thermodynamics & Fluid Mechanics Lab (École polytechnique de Louvain).
Implemented using the Murphy framework — a multiresolution C++ solver for PDEs with octree-based adaptive grids, parallelized via MPI and GPU acceleration.

Parallel Conjugate Gradient Solver

Implementation of a parallelized conjugate gradient method for solving large sparse linear systems, featuring both an MPI-based CPU version and a CUDA GPU version. Benchmarks highlight differences in scalability and performance.
Repository →

Sparse Diffusion Solver with Finite Difference Methods

Explicit time integration of the 2D heat equation using finite difference schemes. Features both Dirichlet and Neumann boundary handling and constructs the Laplacian using sparse matrices. Includes visualization and LU decomposition for validation.
Repository →

Shallow Water Equation Solver

Finite Volume Method solver for the 2D shallow water equations. Implements conservative schemes and boundary treatment for simulating wave propagation and hydraulic jumps.
