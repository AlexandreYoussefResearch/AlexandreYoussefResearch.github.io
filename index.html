Parallel Conjugate Gradient Solver

Parallel implementation of the Conjugate Gradient algorithm
This project compares two high-performance implementations for solving large symmetric positive-definite systems:
- A distributed-memory CPU version using MPI
- A GPU-accelerated version using CUDA
Both versions are benchmarked for scalability and speed.

Shallow Water Equation Solver

1D Finite Volume Simulation of the Saint-Venant Equations
Numerical solution of shallow water equations using two classical schemes: Lax-Friedrichs and Godunov with Roe linearization. The solver compares accuracy and stability across discontinuities.
